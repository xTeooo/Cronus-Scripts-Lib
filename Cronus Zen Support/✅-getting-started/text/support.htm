Cronus Support Checklist

You've likely been given this page because you need help but haven't provided enough information to answer your question, or that you need to verify if all your equipment is working correctly.

HARDWARE TEST

First, let's make sure all your hardware is working correctly, including the Zen, all it's buttons and USB ports, Zen's OLED screen, operating system and all 8 of the memory slots, all 3 of the USB cables provided with the Zen (or any others that you may be using) and finally the USB ports and drivers on your Windows PC.

Check the Cronus Hardware Tests page for complete instructions.

ASKING FOR HELP

Let's start by showing you how NOT to ask for help

"my controller is disconnecting" "ps5 games are not working" "it's not working" "it says device disconnected" "anti-recoil isn't working" "my mouse is not moving" "controller buttons are not detecting in the game" "the script isn't loading"

Your first move is to check the Known Issues page to see if your problem has already been reported, then test to see if any of the provided solutions/workarounds solve your problem. You will find all known bugs reported to that page and updated in real-time

If you ever have a support question, we need the following five pieces of information:

  1. The exact controller/mk model you are gaming with and any A1 donor controllers you are using.
  2. The exact console/pc model that you are gaming on and the game you are trying to play.
  3. A link to the exact instructions you're following.
  4. A screenshot of Zen Studio's Device Panel with everything connected (you can use Windows Snippet or Gyazo - great free tools for screenshots).
  5. A description of your problem so that it can be reproduced. Be sure to use the correct button/label names.

If you have a GamePack question, be sure to speak to the author in the correct community channels.

﻿

Example: The Apex Legends Weapon AI GamePack was written and maintained by LegitCloudzzz, and he is in our Apex Legends Forum and Discord support channels every day. If you have a script question, speak to the author of the script or post to one of the many number of Forums and Discord script support channels (remember we don't provide scripts, that's the community. We provide the GamePacks.

EXAMPLE SUPPORT QUESTIONS

Without your basic setup information, it's impossible to answer any support questions. If you are directed to this page, don't be offended, we simply need you to follow the instructions, so you have a fun and enjoyable experience with your Zen.

﻿BAD﻿

It could be hours or days before you get your answer...

Question: "Hi, my PS5 controller keeps disconnecting."

Answer: "Hi, please read the Cronus Support Checklist then get back to me."

﻿GOOD﻿

Immediate answer and solution...

Question: "Hi, I'm using the latest Zen Studio with PS4 Speciality Enabled. A PS5 DUALSENSE controller is connected to the A1 USB port, a HORI Mini-Pad license donor controller is connected to the A3 USB port and the Zen is connected to a PlayStation 5 from the CONSOLE/PC USB port (micro-USB). I am trying to play Call of Duty Vanguard, but it keeps giving me an error message that my controller is not supported and keeps disconnecting." Answer: "Hey, first up here is the exact step-by-step user guide you need for using the PS4 SPECIALITY method, while connecting your controller to the Zen by USB. The license donor should be connected to A1, the DUALSENSE should be connected to A3, however Call of Duty Vanguard does not support the PS4 SPECIALITY method, so you need to use the PS REMOTE PLAY method instead."
