Programmer Pane - All eight memory slots loaded with GamePacks

The Programmer panel gives easy access to all GamePacks, GPC Scripts, and Remapping Profiles. Cronus Zen has 8 x 32k memory slots and you can make any combination of saves including the full CronusMAX PLUS GamePack library as well as Cronus Zen GamePacks with the new OLED Menu System (GamePacks are ready to use MODs that can setup without any programming knowledge), Remapper Profiles (created with the MAX Mapper plugin), GPC Scripts downloaded from the Library), or GPC script files (your saved scripts files).

Programmer Panel Control Icons

Programmer Control Icons

File Directory

A fully-featured file navigation interface. Includes the option to set your favorite folder. Right-click on the Favorite Folder bar to reset to defaults (Desktop). Also includes a 'Refresh Local Directory' icon which will take you to your favorite folder.

CronusMAX PLUS GamePack Library

This is the entire CronusMAX GamePack library made available to Zen users. Most of these will eventually be ported to the Zen OLED GamePack system. Includes the option to add packs to a Favorites tab. There's also a Newest Tab that shows the very latest uploaded packs. Simply drag and drop to a memory slot.

Cronus Zen GamePack Library

These are the brand new Cronus Zen GamePacks, Pro Packs designed by the Collective Minds Dev Team, Community Packs designed by expert gamers in the Cronus Community. Each pack includes advanced mods and a brand new OLED Menu Interface. Includes the option to add packs to a Favorites tab. There's also a Newest Tab that shows the very latest uploaded packs. Simply drag and drop to a memory slot.

GPC Scripts

Any scripts that you have open in the compiler tab will appear here. Simply drag and drop to a memory slot.

Remapper Profiles

A list of common remapper profiles that allow you to you easily remap all buttons, analog sticks, and sensors to accurately customize the controller layout of any game.

Program Device

When you are ready to program the memory slots to your Cronus Zen simply click on the Program Device Icon.

Reload Memory Slots

If Zen is connected to your PC and you modify a mod via the GamePack Configuration Interface (GCI), it's advisable to click Reload Memory Slots so that Zen Studio can read back the new data correctly.

Delete Memory Slots

Drag and drop any memory slot into the trash can to empty the memory slot. Click Program Device once you are ready to erase or overwrite.

Memory Slots

GPC Scripts, Remapper Profiles, and GamePacks can be loaded to each memory slot. GamePacks have extra functionality, as shown by three icons.

UPD is displayed if there has been an update to a GamePack that you had previously installed to a memory slot. Click on the icon to instantly update the slot ready for configuration and programming.

ZEN is displayed if the GamePack is designed for the enhanced capabilities of the Cronus Zen. You can also click on this to load the GamePacks instructions and support.

NUMBER is the memory slot number. You can also click on this to load the GamePack Configuration Interface (GCI).

How to Download & Program a Script from the GPC Library

Scripts downloaded from the library are written and maintained by members of the community. Each will have it's own instructions and methods of operation. All are different so we recommend you read any notes and instructions which are usually in the header of the script. If you need further help you can click on the Authors name and contact them directly through the Cronus Community.

How to Download & Program a Zen GamePack

GamePacks are pre-configured MODS written by expert gamers in the Cronus Community and also from the development team at Collective Minds which are marked as [PRO]. All Zen GamePacks include an easy-to-use setup interface both in Zen Studio and on the OLED screen of your Cronus Zen.

Tip: It is highly recommended that you first try the FPS Master Zen GamePack as this is quite easy to set up as the defaults are all pretty strong with most FPS Games and it gives you every major MOD available in one pack - it's a great starter for anyone new to how GamePacks work.

Once FPS Master is programmed, select the correct memory slot by pressing the P1 button on your Zen (Memory Slot 1 in the example below) and then press the LT/L2 + MENU/OPTIONS buttons on your controller to enter the OLED Menu System.
