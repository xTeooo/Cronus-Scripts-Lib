Some PC games may not play or display a warning

If you get an error saying that the game will not boot or some other warning, it means some of your settings are not correct. This is how you should setup any controller on a Windows PC. For this example, we have used an original Xbox controller (any original model will work), and many licensed third-party controllers will also work. Be sure you have updated Zen Studio and Zen Firmware to the latest versions:

Zen Studio: Zen Firmware: ï»¿

ï»¿IMPORTANT: Make sure your Xbox controller is fully charged and close any game you are trying to play while setting up your Zen on a Windows PC.

  1. Connect the CRONUS ZEN to a PC using both the PROG (mini-USB) and CONSOLE/PC (micro-USB).
  2. Open ZEN STUDIO > Device Panel > Emulator Output Protocol set to AUTO. Make sure that PS4 Speciality and PS Remote Play are NOT enabled.
  3. ï»¿IMPORTANT: Close Zen Studio and unplug the PROG (mini-USB) cable from the Zen.
  4. Connect an XBOX controller to A1 and press the XBOX button to power it on.
  5. Tap Zen's reset button to make sure everything syncs up correctly, and it's a good idea to visit gamepad-tester.com to make sure all the controller outputs are being correctly detected by Windows.
If successful, the Zen RGBs will change to solid GREENï»¿ï»¿, and the XBOX LED will stop flashing and change to solid WHITE, indicating that Zen, controller and PC are now correctly paired. You can now boot the game you wish to play. If it's not working, make sure the controller is powered on by pressing the XBOX button, then press Zen's reset button a couple of times and that should re-sync everything. Also make sure the controller is connected to A1 and that the output is set to AUTO. Do not waste your time setting it up any other way.

ï»¿AUDIO: If you wish to use a headset, connect it directly to the PC's sound card and not the controller. If you don't have a headset and wish to use voice chat, you can also use a laptop's built-in microphone.

PS Remote Play Lag

If you experience lag with the official PS Remote Play app, try Chiaki PS Remote Play , a very popular PS Remote Play alternative that is fast and easy to setup. Many community members have also offered the following tips for lag free PS5 gaming:

  1. Your console and PS Remote device should both be on the same wired network.
  2. If you are using a modern USB-C mobile device, use a USB-C GIGABIT ethernet adapter.
  3. Upload speed should be at least 5Mbps - but the higher the better.
  4. Ping/Latency is super important. Anything below 20ms is good. Bad ping = ğŸ’€
  5. Try another device - PS Remote Play may not work well for you on a Mac, but Android Chiaki may be the ticket.

Xbox Bluetooth

If your Bluetooth model Xbox controller is updated to 5.13.3146.0 or above, it no longer supports Bluetooth. If you wish to use your Xbox controller wirelessly with the latest controller firmware, you can still use an Xbox Wireless Adapter.

Affected Xbox controllers are:

  • Xbox One 1708
  • Xbox Elite S2 1797
  • Xbox Series X/S 1914

Audio Issues

While Zen supports wired headsets in multiple configurations including crossover, you should know that audio is heavy on CPU cycles.

This isn't a problem if you do not use GamePacks or scripts, but if you are using a powerful GamePack or script, audio may affect its performance by causing the controller to disconnect or introduce lag.

Read the Wired Headset Users page for more details.

It is highly recommended that if you are using MODs, that you also use a wireless headset or any headset that connects to the console and not the controller. When gaming on a PC, connect the headset directly to your computers sound card.

PROBLEM? WHAT WE NEED...

If you ever have a support question, we need the following five pieces of information:

  1. The exact controller/mk model you are gaming with and any A1 donor controllers you are using.
  2. The exact console/pc model that you are gaming on and the game you are trying to play.
  3. A link to the exact instructions you're following.
  4. A screenshot of Zen Studio's Device Panel with everything connected (you can use Windows Snippet or Gyazo - great free tools for screenshots).
  5. A description of your problem so that it can be reproduced and be sure to use the correct button/label names (see below).

If you have a GamePack question, be sure to speak to the author in the correct community channels.
