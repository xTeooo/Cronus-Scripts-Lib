Everything you need to program the Cronus Zen firmware, including step instructions for both downgrading and upgrading. Note: Since Zen Studio Build 1.2.1-beta.105, you no longer need to download the firmware update tool separately - it is now built into Zen Studio (Tools > Firmware Update).

UPGRADING

Upgrading the firmware is quite simple and takes less than 20 seconds. You will need a micro-USB cable connected to the CONSOLE/PC USB port (two are provided with the Zen).

Zen CONSOLE/PC USB PORT to PC (micro-USB)
  1. Open Zen Studio and select Tools > Firmware Update
  2. Connect Zen to a PC from the CONSOLE/PC USB PORT (micro-USB cable).
  3. Hold down the small blue reset button until the OLED Screen displays Zen Bootloader
  4. If detected correctly, the firmware update tool will display Collective Minds Cronus ZEN, the serial number, and the firmware version. NOTE: If it doesn't detect try another micro-USB cable.
  5. Click on the detected Zen hardware and you'll now see a firmware selection screen with a blue toggle labeled Use the Latest Version Available.
  6. Click Continue > Begin Update to update to the latest version (as of today it's ).
  7. Once the firmware upgrade has successfully been completed, the Zen's RGB LEDs will flash RED, GREEN, and BLUE. and you are ready to setup Zen Studio.

DOWNGRADING

You need to downgrade the firmware if Zen is ever stuck with an E3 error code.

  1. Open Zen Studio and select Tools > Firmware Update
  2. Connect Zen to a PC from the CONSOLE/PC USB PORT (micro-USB cable).
  3. Hold down the small blue reset button until the OLED Screen displays Zen Bootloader
  4. If detected correctly, the firmware update tool will display Collective Minds Cronus ZEN, the serial number, and the firmware version. NOTE: If it doesn't detect try another micro-USB cable.
  5. Click on the detected Zen hardware and you'll now see a firmware selection screen with a blue toggle labeled Use the Latest Version Available.
  6. Click the blue toggle so that it slides to the left, revealing all available firmware versions, from the oldest to the newest.
  7. Scroll to the bottom of the list, select firmware version , then click Continue > Begin Update. Once the firmware downgrade has successfully been completed, the Zen's RGB LEDs will flash RED, GREEN, and BLUE.
