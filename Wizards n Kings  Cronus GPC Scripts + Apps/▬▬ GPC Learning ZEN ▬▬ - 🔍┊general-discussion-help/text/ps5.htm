PlayStation 5

Cronus Zen supports all versions of the PlayStation 5 console. Connect your favorite PlayStation, Xbox, and Nintendo controller, including Mouse & Keyboard support, all with access to the famous Cronus GamePacks and GPC script engine.

ï»¿

OK great, we have your attention!

ï»¿

The Cronus Support Center, Discord, Forums, and every comment section you can think of, has a whole bunch of Zen users who are completely confused by how PS5 support works, what the different methods are, and knowing when and how to use them. Unfortunately, they all have one thing in common - they didn't read this page of the guide yet :)

ï»¿

So here it is, playing PS5 games requires just 2 methods. PS4 SPECIALITY or PS REMOTE PLAY. There is no third method, it's one or the other. Play a game without its correct method and your controller will disconnect every 8-10 minutes, disconnect straight away, or not boot the game at all.

ï»¿

The good news is, knowing which method to use, and then setting it up, couldn't be easier! If you get the following error, or one like it when booting a PS5 game, then it will not support the PS4 SPECIALITY method, so instead you must use the PS REMOTE PLAY method.

ï»¿

ï»¿

You can scroll down for more details, but that's pretty much all there is to it. As soon as you know which method to use for your PS5 game, simply navigate to the correct PS4 SPECIALITY or PS REMOTE PLAY instructions and you're good to go! Note: PS4 games will always use the PS4 SPECIALITY method.

PS4 SPECIALITY?

We recommend you test every game you have, both PS5 and PS4 versions, using PS4 SPECIALITY first, so that you know for sure if it will work or not. Many PS5 games work with this method, as well as most if not all PS4 games - Click here for the current known list of compatible PS5 games.

It is easy to find out if the game you are trying to play works with PS4 SPECIALITY or not. If the PS5 game fails to boot and you see an error saying that the controller is not compatible or a message that the controller has disconnected, it will NOT work with the PS4 SPECIALITY method, so you must use the PS REMOTE PLAYï»¿ï»¿ method instead.

PS REMOTE PLAY?

The only time you need to use the PS REMOTE PLAY method is when the game does not support PS4 SPECIALITY. That's the beginning and end of it.

It is easy to find out if the game you are trying to play requires PS REMOTE PLAY or not. If the PS5 game fails to boot and you see an error saying that the controller is not compatible or a message that the controller has disconnected, then the only supported method for that game is PS REMOTE PLAY.

It requires a good quality internet connection, preferably wired ethernet, and any PC/Mac/iOS/Android device to run the official SONY PS REMOTE PLAY app, or the awesome open-source CHIAKI REMOTE PLAY for both Windows and Android devices.
