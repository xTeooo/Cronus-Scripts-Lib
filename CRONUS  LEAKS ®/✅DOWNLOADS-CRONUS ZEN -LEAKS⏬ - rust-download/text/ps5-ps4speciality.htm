PS4 SPECIALITY

Connect any supported PlayStation, Xbox, and Nintendo controller to a PS5 using the PS4 SPECIALITY method, all with full MOD & GPC scripting capabilities.

Playing PS5 games requires just 2 methods. PS4 SPECIALITY or PS REMOTE PLAY. There is no third method, it's one or the other. Play a game without its correct method and your controller will disconnect every 8-10 minutes, disconnect straight away, or not boot the game at all.

ï»¿

If when booting a PS5 game and you get an error like the one below, PS4 SPECIALITY will not work with that game, so you will need to use the PS REMOTE PLAY method instead.

Note: Most if not all PS4 games will work with the PS4 SPECIALITY method.

PS4 SPECIALITY DONOR CONTROLLERS

This is a list of known working third-party licensed PS4 controllers that are compatible as a PS4 SPECIALITY license donor controller. The donor device must be connected to A1 at all times (or A2 if using a wired headset to a wired DUALSENSE controller). The controller you wish to game with is then connected to A3 (or a Wireless Adapter/Bluetooth, depending on the controller).

For your convenience, we've created a Google Sheet that lists all currently known PS5 games that support / don't support the PS4 SPECIALITY method.
